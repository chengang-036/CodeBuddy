# 危老板的衣橱管理系统

智能衣服管理与搭配系统，支持本地存储和云端同步两种模式。

## 🚀 快速开始

### 本地版（无需配置，立即使用）

适合单设备使用，数据存储在浏览器本地。

- **衣服上架**: 打开 `upload.html`
- **衣橱管理**: 打开 `wardrobe.html`
- **衣服搭配**: 打开 `match.html`
- **搭配方案**: 打开 `outfits.html`

### 云端版（跨设备同步）⭐ 推荐

支持多设备数据同步，手机、电脑、平板数据实时共享。

#### 第一步：配置 Firebase

1. 阅读 `FIREBASE_SETUP.md` 完整教程
2. 创建 Firebase 项目（5 分钟）
3. 配置 `firebase-config.js` 文件

#### 第二步：使用云端版

- **衣服上架**: 打开 `upload-firebase.html`
- **衣橱管理**: 打开 `wardrobe-firebase.html`

## 📱 功能特点

### ✨ 本地版特点
- ✅ 无需配置，开箱即用
- ✅ 完全离线使用
- ✅ 数据保存在浏览器
- ⚠️ 仅限单设备访问

### ☁️ 云端版特点
- ✅ 跨设备数据同步
- ✅ 手机、电脑实时共享
- ✅ 数据永久保存
- ✅ 支持图片云端存储
- ⚠️ 需要 5 分钟配置

## 🆚 两个版本对比

| 功能 | 本地版 | 云端版 |
|------|--------|--------|
| 配置难度 | ⭐ 无需配置 | ⭐⭐⭐ 需配置 Firebase |
| 跨设备访问 | ❌ 不支持 | ✅ 支持 |
| 数据同步 | ❌ 无法同步 | ✅ 实时同步 |
| 存储容量 | 5-10 MB | 5 GB |
| 离线使用 | ✅ 完全离线 | ⚠️ 需要网络 |

## 💡 使用建议

**如果您：**
- 只在一台设备使用 → 选择**本地版**
- 需要手机、电脑同步 → 选择**云端版** ⭐
- 想要数据永久保存 → 选择**云端版** ⭐

## 📚 详细文档

- **Firebase 配置教程**: `FIREBASE_SETUP.md`
- **项目主页**: 打开 `index.html`

- **前端**: HTML5、CSS3、JavaScript
- **本地存储**: localStorage
- **云端存储**: Firebase Firestore + Firebase Storage
- **图片压缩**: Canvas API
