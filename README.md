# CodeBuddy

CodeBuddy 练习项目集合

## 项目列表

### 0001-HelloWorld
一个简单的 Hello World 网页示例，展示了基础的 HTML、CSS 和动画效果。

**特点：**
- 渐变背景
- 毛玻璃效果
- 淡入动画

**查看方式：**
直接在浏览器中打开 `0001-HelloWorld/index.html` 文件即可。

### 0002-Claim (寿险报案系统)
完整的寿险报案信息收集系统，支持动态表单、数据存储、记录查询等功能。

**功能特点：**
- 📝 智能表单：根据选择动态显示/隐藏相关字段
- 💾 本地存储：使用 localStorage 保存报案数据
- 🔍 记录查询：搜索、查看详情、删除记录
- 📊 统计面板：总报案数、今日报案数
- 📤 数据导出：支持导出 JSON 格式数据
- 📱 响应式设计：支持手机、平板、电脑访问

**包含页面：**
- `index.html` - 报案表单页面
- `records.html` - 记录查询页面

**查看方式：**
直接在浏览器中打开 `0002-Claim/index.html` 文件即可。
